Vue d'ensemble de Meta-Copy

Qu'est-ce que Meta-Copy?

Meta-Copy est un script pour l'extension "User JavaScript and CSS" qui améliore l'expérience utilisateur sur Metamob.fr. Il ajoute un bouton de copie à côté du nom de chaque monstre et des indicateurs visuels pour faciliter le suivi des archimonsters que vous avez copiés.

Pourquoi l'utiliser?

  • Copiez les noms d'archimonsters en un seul clic au lieu de les sélectionner manuellement
  • Indicateurs visuels montrant le dernier monstre copié (bouton qui devient une coche ✅)
  • Les noms avec cases blanches dans la section Trade "Communauté" deviennent cliquables pour copie rapide
  • Mise en évidence en rouge du dernier nom copié dans la section Trade pour un suivi facile

Questions Fréquentes

L'utilisation de ce script est-elle autorisée par Metamob.fr?

Oui, ce script est simplement une amélioration de l'interface utilisateur qui facilite la copie des noms. Il ne modifie pas le contenu ou les fonctionnalités de base du site, il ne fait qu'ajouter des boutons et des indicateurs visuels pour améliorer l'expérience utilisateur.

Comment installer l'extension "User JavaScript and CSS"?

L'extension est disponible pour la plupart des navigateurs populaires. Vous pouvez la trouver dans le Chrome Web Store, Firefox Add-ons, ou autres magasins d'extensions selon votre navigateur. Une fois installée, vous pourrez ajouter notre script spécifique pour Metamob.fr.

Le script fonctionne-t-il avec tous les navigateurs?

Le script fonctionne avec tous les navigateurs qui supportent l'extension "User JavaScript and CSS", notamment Chrome, Firefox, Edge et Opera. Les instructions d'installation spécifiques sont fournies après l'achat.

Dois-je configurer quelque chose après l'installation?

Non, le script fonctionne automatiquement dès son installation. Il n'y a pas de paramètres supplémentaires à configurer. Vous verrez immédiatement les boutons de copie apparaître à côté des noms d'archimonsters sur Metamob.fr.

Comment puis-je obtenir de l'aide en cas de problème?

Notre équipe de support est disponible via Discord. Nous vous aidons avec l'installation, les mises à jour ou tout autre problème que vous pourriez rencontrer avec le script. Des guides détaillés sont également fournis après l'achat.
